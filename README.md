# TurathAI Frontend

## 🚀 Overview
TurathAI Frontend is the Angular-based user interface for the **TurathAI** platform, an AI-driven initiative to promote sustainable tourism by showcasing Tunisia’s rich heritage. This frontend interacts with the **TurathAI Backend** to provide a seamless user experience.

---

## ✨ Features
- 🎨 **User-Friendly Interface**: Responsive and intuitive design.
- 🔐 **Authentication & Authorization**: Secure login and role-based access control.
- 🏛 **Heritage Site Exploration**: View and interact with detailed heritage site information.
- 🎉 **Event Management**: Browse and participate in cultural events.
- ⭐ **Review System**: Leave ratings and reviews for heritage sites.
- 🤖 **AI-Powered Recommendations**: Personalized suggestions based on user preferences.
- 🎮 **Gamification**: Engage users with badges, points, and rewards.

---

## 🛠 Technologies Used
- **Frontend Framework**: Angular
- **UI Library**: Angular Material / TailwindCSS
- **State Management**: NgRx (optional)
- **API Communication**: RESTful APIs via HttpClient
- **Authentication**: JWT-based authentication
- **Version Control**: Git, GitHub
- **Build & Deployment**: Angular CLI, Docker, AWS (optional)

---

## 🚀 Getting Started

### ✅ Prerequisites
Ensure you have the following installed:
- Node.js (LTS version recommended)
- Angular CLI (`npm install -g @angular/cli`)
- Git

### 📥 Installation
1. **Clone the Repository**:
   ```bash
   git clone https://github.com/MarwenDev2/TurathAI-Frontend.git
   cd TurathAI-Frontend
   ```
2. **Install Dependencies**:
   ```bash
   npm install
   ```
3. **Run the Development Server**:
   ```bash
   ng serve
   ```
4. **Access the Application**: Open `http://localhost:4200/` in your browser.
